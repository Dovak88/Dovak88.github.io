Agapé

Mobile game
Music – Sound Design

Agapé is a mobile game made by a team of Isart’s students as part of a game jam (GameWeek 2020)

We had 4 days to make this game from start to finish. In this project I did the sound design and the music composition.

It was a real challenge because I had a lot of small musical segments to make in order to illustrate the feelings of the main character who is going through a depressive episode.
