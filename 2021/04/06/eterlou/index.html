• Eterlou

2D Platformer game

Unity – FMOD

Eterlou is a platformer game made by a team of Isart’s students.

We had a month to make a 2D-platfomer game from start to finish. In this project I did the sound design, music composition and FMOD integration. I took an extra step and worked closely with the developers by handling the audio integration in unity myself. In order to achieve that I made an audio manager and some other audio related scripts  in c#. This allowed the developers to focus more on physics and graphic integration.
